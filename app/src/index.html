Which sides do we care about? Which sides should be monitored?

For all sides that are selected, elements must be completely within those sides to be considered "in" the viewport.

If a side is unselected, then withinViewport will ignore whether an element runs out of that side, and the element will still be considered "in".

Typically, all sides should be selected.
